## Introdução 🎯
O **Conhecendo Band** é um aplicativo que reúne informações sobre pontos turísticos, bares, hotéis, restaurantes e eventos da cidade. Com uma navegação simples e leve, ele busca auxiliar turistas e moradores locais na descoberta de novos lugares.

## Funcionamento ⚙️
Para rodar o aplicativo em seu ambiente de desenvolvimento, siga os seguintes passos:

1. Baixe o arquivo `.rar` do código e extraia-o.
2. Abra o terminal na pasta extraída.
3. Instale as dependências do projeto com um dos seguintes comandos, dependendo do seu gerenciador de pacotes:
   - `npm install`
   - `yarn install`
4. Se estiver utilizando o **Android Studio**, abra um dispositivo (emulador) por ele.
5. Caso prefira testar no seu **próprio celular**, baixe o aplicativo **Expo Go** na loja de aplicativos.
6. No terminal, execute o comando `expo start` e selecione a opção desejada para rodar o aplicativo.
